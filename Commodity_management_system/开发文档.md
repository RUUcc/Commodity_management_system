## 六、核心业务功能

### 6.1 用户登录流程

```
GET /login → 显示登录页
POST /doLogin → 用户名+密码 → UserService.login()
    ├── LambdaQueryWrapper 查用户
    ├── BCryptPasswordEncoder.matches() 比对密码
    ├── 成功 → Session.setAttribute("loginUser", user) → redirect /index
    └── 失败 → model.addAttribute("error", "用户名或密码错误") → 返回 login.html
GET /logout → session.invalidate() → redirect /login
```

### 6.2 登录拦截器

```
LoginInterceptor.preHandle()
    ├── Session 中有 loginUser → 放行
    └── Session 中无 loginUser → response.sendRedirect("/login")

放行路径: /login, /doLogin, /css/**, /js/**, /uploads/**
```

### 6.3 商品管理（核心业务）

| 功能 | URL | 方法 | 说明 |
|------|-----|------|------|
| 分页列表 | `/product/list` | GET | Thymeleaf 渲染，每页 5 条 |
| 多条件查询 | `/product/list?name=&category=` | GET | LambdaQueryWrapper 动态拼接 |
| 新增商品 | `/product/add` | POST | @Valid 校验 + 图片 MultipartFile 上传 |
| 修改商品 | `/product/edit` | POST | @Valid 校验 + 回显 + 重新上传图片 |
| 删除商品 | `/product/delete?id=` | GET | @CacheEvict 清除缓存 |
| 查询单个 | `/product/get?id=` | GET | 编辑回显用 |

### 6.4 缓存策略

```
ProductServiceImpl:
    查询 pageQuery()         → @Cacheable(value="product", key="...")
    新增 save()              → @CacheEvict(value="product", allEntries=true)
    修改 updateById()        → @CacheEvict(value="product", allEntries=true)
    删除 removeById()        → @CacheEvict(value="product", allEntries=true)
```

### 6.5 文件上传

```
配置文件:   D:/spring-boot/期末设计/uploads/
URL 映射:   /uploads/xxx.jpg → 本地文件
实现方式:   WebConfig.addResourceHandlers() 注册静态资源映射
唯一命名:   UUID + 原文件后缀
```

## 七、安全与规范实现

### 7.1 BCrypt 密码加密

```
SecurityConfig.java        → 注册 BCryptPasswordEncoder Bean
UserServiceImpl.save()     → password = passwordEncoder.encode(password)
UserServiceImpl.updateById() → 传了新密码则加密，否则保留原密码
UserServiceImpl.login()    → passwordEncoder.matches(password, dbPassword)
DataInitializer.java       → 启动时检测明文密码并自动升级
```

### 7.2 @Valid 参数校验

| 实体字段 | 校验注解 |
|---------|---------|
| `User.username` | `@NotBlank(message = "用户名不能为空")` |
| `User.password` | `@NotBlank(message = "密码不能为空")` |
| `Product.name` | `@NotBlank(message = "商品名称不能为空")` |
| `Product.price` | `@NotNull` + `@DecimalMin("0.01")` |
| `Product.stock` | `@Min(0)` |

### 7.3 全局异常处理

```java
@RestControllerAdvice
public class GlobalExceptionHandler {
    BindException.class        → "商品名称不能为空；价格不能为空"
    IllegalArgumentException.class → e.getMessage()
    Exception.class            → "系统内部错误，请联系管理员"
}
```

### 7.4 @Transactional 事务

```
ProductServiceImpl.save()
ProductServiceImpl.updateById()    @Transactional(rollbackFor = Exception.class)
ProductServiceImpl.removeById()
UserServiceImpl.save()
UserServiceImpl.updateById()
```

## 八、MyBatis 使用方式（双模式）

### 方式一：MyBatis-Plus 增强

```java
// BaseMapper 自带 CRUD
this.baseMapper.insert(entity);
this.baseMapper.deleteById(id);
this.baseMapper.updateById(entity);

// LambdaQueryWrapper 条件构造器
LambdaQueryWrapper<Product> wrapper = new LambdaQueryWrapper<>();
wrapper.like(StringUtils.hasText(name), Product::getName, name)
       .eq(StringUtils.hasText(category), Product::getCategory, category);

// 分页查询
Page<Product> page = new Page<>(pageNum, pageSize);
this.page(page, wrapper);
```

### 方式二：原生 MyBatis XML 映射

```xml
<!-- resources/mapper/ProductMapper.xml -->
<select id="selectByCondition" resultMap="ProductResultMap">
    SELECT * FROM tb_product
    <where>
        <if test="name != null and name != ''">
            AND name LIKE CONCAT('%', #{name}, '%')
        </if>
        <if test="category != null and category != ''">
            AND category = #{category}
        </if>
        <if test="minPrice != null">
            AND price >= #{minPrice}
        </if>
        <if test="maxPrice != null">
            AND price <= #{maxPrice}
        </if>
    </where>
    ORDER BY create_time DESC
</select>
```

### 方式三：MyBatis 注解

```java
@Select("SELECT * FROM tb_product WHERE category = #{category} ORDER BY create_time DESC")
List<Product> selectByCategory(@Param("category") String category);

@Update("UPDATE tb_product SET stock = stock - #{count} WHERE id = #{id}")
int reduceStock(@Param("id") Integer id, @Param("count") Integer count);
```

## 十二、踩坑记录

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| `PaginationInnerInterceptor` 找不到 | MyBatis-Plus 3.5.9 拆分了分页模块 | 添加 `mybatis-plus-jsqlparser` 依赖 |
| `CannotGetJdbcConnectionException` | MySQL 8.0 `caching_sha2_password` 认证 | 连接 URL 加 `allowPublicKeyRetrieval=true` |
| Thymeleaf `#temporals.format` 报错 | `java.util.Date` 不兼容 | 改用 `LocalDateTime` |
| `th:onclick` 报 TemplateProcessingException | Thymeleaf 3.1 安全限制 | 改用 `th:attr="data-id=${...}"` + `this.dataset.id` |
| `BCryptPasswordEncoder` 找不到 | 缺少独立加密包 | 添加 `spring-security-crypto` 依赖 |
| `Result` 导入缺失 | 文件重写时丢失 import | 补充完整 import 语句 |
