# Commodity_management_system
本设计基于Spring Boot框架，结合MyBatis-Plus ORM技术与Thymeleaf模板引擎，设计并实现了一个具有商品信息管理、用户管理、文件管理、登录认证等功能的商品管理系统。系统前端采用Thymeleaf模板引擎结合HTML5+CSS3+JavaScript构建响应式页面，后端使用Spring Boot 3.0.2作为核心框架、MyBatis-Plus 3.5.9作为持久层框架、MySQL作为关系型数据库。系统实现了商品的增删改查、多条件模糊搜索、分页显示、图片上传、用户登录认证与会话管理、文件资源管理等功能模块，并集成了Caffeine本地缓存和Spring Cache抽象以优化查询性能。
